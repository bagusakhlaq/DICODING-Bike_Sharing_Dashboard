# Final Project: Bike Sharing Dashboard :bike:

## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install pandas numpy matplotlib seaborn streamlit
```

## Run steamlit app
```
py -m streamlit run dashboard.py
```
